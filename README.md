## Description
Ticket bot

## For start
### Installation

```bash
$ npm install
$ yarn
$ pnpm install
```

### Running the app

```bash
# development
$ npm run start
$ yarn run start
$ pnpm run start

# watch mode
$ npm run start:dev
$ yarn run start:dev
$ pnpm run start:dev

# production mode
$ npm run start:prod
$ yarn run start:prod
$ pnpm run start:prod
```

## Test

```bash
$ pnpm run test
$ pnpm run test:e2e
$ pnpm run test:cov
```
